# Описание
Одностраничное приложение на React.js для поиска книг в сервисе Google Books.

## Дополнительные библиотеки
* Redux и React Redux. Использовались для управления состояния приложения. Например, индикация загрузки книг через API или контроль текущих данных, которые поступили через это API. 
* React Router v6. Использовалась для создания навигации между главной страницей, содержащей поиск, и отдельными страницами для каждой книги.
* html-react-parser. Использовался для безопасного парсинга html-строки с описанием книги, которую возвращало API.
* MUI. Библиотека была использована для создания интерфейса сайта в стиле сервисов Google.

## Отмеченные проблемы
* Google Books API работает достаточно странно, в зависимости от пагинации запроса ответы могли приходить несколько разными. Например, указывалось разное количество всех найденных книг по запросу.
* Странный баг с React списком, когда в консоль выводилась информация об ошибках, что есть несколько элементов с одним ключом. Ключ напрямую брался как уникальный id книги из API. Данный баг на работоспособности приложения никак не сказывается.
